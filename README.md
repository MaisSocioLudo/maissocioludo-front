# +SocioLudo
Este repositório contém o site de apoio didático do projeto **SocioLudo Digital**, desenvolvido como parte do Trabalho de Conclusão de Curso do **Curso Técnico em Informática do Instituto Federal de Educação, Ciência e Tecnologia do Rio de Janeiro — Campus Arraial do Cabo**.

O projeto faz parte do TCC:

**Aperfeiçoamento da Didática do Jogo SocioLudo Digital**

Desenvolvido por:

* Beatriz Vianna
* Flávio Soares
* Gabriela Almeida
* Júlia Martins de Oliveira

## Sobre o projeto

O **SocioLudo Digital** é uma versão computacional de um jogo didático voltado para o ensino de Sociologia na Educação Básica. A proposta do jogo é tornar o aprendizado mais acessível, interativo e dinâmico, utilizando recursos digitais e elementos de gamificação para aproximar os estudantes dos conteúdos sociológicos.

Este site foi desenvolvido como uma extensão pedagógica do jogo, funcionando como um **repositório web de materiais didáticos gratuitos e acessíveis**. Nele, os estudantes podem consultar textos de apoio relacionados aos temas presentes nas perguntas do SocioLudo Digital.

## Objetivo do site

O objetivo principal do site é oferecer um ambiente digital de apoio ao aprendizado, permitindo que os alunos aprofundem os conteúdos trabalhados durante as partidas do jogo.

A proposta busca complementar a experiência do SocioLudo Digital por meio de textos didáticos, organizados por temas, com linguagem acessível e foco na compreensão dos conceitos sociológicos.

## Funcionalidades

* Página inicial com apresentação do projeto;
* Listagem de temas sociológicos abordados no jogo;
* Páginas individuais para cada tema;
* Textos didáticos de apoio;
* Organização dos conteúdos por meio de arquivos JSON;
* Estrutura dinâmica para facilitar a criação e manutenção das páginas;
* Interface responsiva e acessível;
* Navegação simples para estudantes e professores.

## Tecnologias utilizadas

O projeto foi desenvolvido utilizando tecnologias modernas de desenvolvimento web:

* **Next.js** — framework React utilizado para criação do site;
* **React** — biblioteca JavaScript para construção da interface;
* **TypeScript** — linguagem baseada em JavaScript com tipagem estática;
* **CSS** — utilizado para a estilização das páginas;
* **JSON** — utilizado para armazenar e organizar os dados dos temas e conteúdos.

## Relação com o TCC

O site faz parte da proposta de aperfeiçoamento pedagógico do SocioLudo Digital.

Durante o desenvolvimento do TCC, foi identificada a necessidade de ampliar o potencial didático do jogo, principalmente por meio de explicações e materiais complementares. Assim, o site foi criado para apoiar o estudante fora do momento da partida, permitindo que ele revise conteúdos, compreenda conceitos e aprofunde os temas sociológicos abordados no jogo.

Dessa forma, o projeto contribui para transformar o SocioLudo Digital em uma ferramenta mais completa para o ensino de Sociologia.

## Público-alvo

O projeto é voltado principalmente para:

* estudantes do Ensino Médio;
* professores de Sociologia;
* escolas públicas;
* pesquisadores interessados em jogos didáticos, gamificação e ensino de Ciências Humanas.
